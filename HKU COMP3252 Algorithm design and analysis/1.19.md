# Divide and conquer
One example **merge sort**

## Sort Problem and Merge Sort Algorithm
round down- 下取整 round up-上取整

Merge sort:
+ Divide step: Divide the whole array into two halves $a_{1:\lfloor n/2\rfloor}$ and $a_{\lceil n/2\rceil+1:n}$
+ Conquer step: sort each recursely
+ Meger step: merge them
